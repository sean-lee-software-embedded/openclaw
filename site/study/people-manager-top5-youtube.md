# People Manager 學習影片 Top 5（YouTube / TED 系）

目標：你即將入職當主管，這份不是泛泛的「勵志清單」，而是從 **帶人、授權、建立信任、提升效率、讓團隊能自主運作** 的角度，挑出最值得看的 5 支影片。

## 我挑選的標準

我優先挑的是：

- 能幫你建立 **people manager 的底層觀念**

## Top 5

## 1. Simon Sinek — Leaders Eat Last / Trust and Leadership（YouTube 多版本）
- 代表連結（YouTube 搜尋結果）：
  - https://www.youtube.com/watch?v=RMfB7MVeeFA
  - https://www.youtube.com/watch?v=WLwFtHnt1gU
- 為什麼值得看：
  - 這支的核心不是「怎麼管人」，而是 **領導者的責任是先照顧團隊**。
  - 如果你想帶人又帶心，這是最重要的底層思維之一。
- 你要特別吸收的點：
  - **信任不是口號，是領導者先給安全感**
  - 團隊不會因為你有 authority 就跟你，而是因為你讓他們感覺安全
  - 授權的前提不是 control，而是 trust + responsibility
- 看完後你應該問自己：
  - 我的團隊會不會因為怕我而不敢講真話？
  - 我有沒有真的替團隊擋壓力，而不是只往下壓？

---

## 2. Amy Edmondson / Google Project Aristotle 相關影片 — Psychological Safety
- 先看觀念：Google Project Aristotle / Psychological Safety 的相關解說影片（YouTube 搜尋：`psychological safety project aristotle youtube`）
- 搭配閱讀概念來源：
  - https://www.shrm.org/executive-network/insights/4-ways-leaders-can-build-psychological-safety-work-teams
- 為什麼值得看：
  - 你想讓團隊自主運作，**心理安全感** 幾乎是必要條件。
  - 沒有心理安全感，大家只會等指令、不敢冒險、不敢報壞消息。
- 你要特別吸收的點：
  - 團隊高績效不只是靠明星成員，而是靠 **能不能安全地講話、問問題、承認錯誤**
  - 主管的情緒反應，會直接決定團隊是否願意說真話
- 你的管理行動：
  - 把錯誤先當作學習素材
  - 開會時主動拉出安靜的人說話
  - 對提問和不同意見，不要第一時間防衛

---

## 3. Delegation — An Essential Trait For Leadership（TED/YouTube 類）
- 代表連結：
  - https://www.youtube.com/watch?v=fFzSNCDO_4w
- 為什麼值得看：
  - 很多新主管最大的坑不是不努力，而是 **不敢授權 / 授權不完整 / 事事自己接**。
  - 你說想讓團隊自主運作，這支正好打到核心。
- 你要特別吸收的點：
  - 授權不是把工作丟出去，是把 **責任、邊界、預期結果** 一起交出去
  - 授權做得好，團隊會長；授權做得差，團隊只會更依賴你
- 實際提醒：
  - 交辦時不要只講 task，要講 **目的、品質標準、時間點、可自行決策範圍**
  - 不要授權後又 micromanage

---

## 4. Mary Steffel — What You Gain by Giving Choices Away
- 連結：
  - https://www.ted.com/talks/mary_steffel_what_you_gain_by_giving_choices_away
- 為什麼值得看：
  - 這支對「授權」有一個很好的補充：
  - **真正的領導，不是所有決定都抓在手上，而是知道哪些決定該讓出去。**
- 你要特別吸收的點：
  - 放掉一部分控制，不等於失控
  - 給團隊選擇權，會增加 ownership
- 對你有用的地方：
  - 當主管後你不會缺工作做，你真正缺的是：
    - 哪些要自己抓
    - 哪些要交出去
    - 哪些要讓團隊自己決定

---

## 5. Feedback / Performance Conversation 類 TED / 管理影片
- 可先看這支清單導讀，再挑 1–2 支最貼近你的：
  - https://www.cultureamp.com/blog/ted-talks-performance-review
- 相關 YouTube/TED 搜尋關鍵字：
  - `giving feedback ted youtube`
  - `constructive feedback manager youtube`
- 為什麼值得看：
  - 當主管後，你不是只分配工作，還要 **給回饋、拉人、糾偏、激勵**。
  - 很多主管帶不好人，不是因為不 caring，而是不會 feedback。
- 你要特別吸收的點：
  - 回饋不是年底才做
  - 回饋不是情緒發洩
  - 回饋是幫對方變強
- 你應該練的能力：
  - 明確指出行為與影響
  - 不羞辱人
  - 讓對方知道如何改進
  - 讓對方感覺被支持，不是被審判

# 你當 People Manager 最值得先記住的 5 句話

1. **你不是來證明你最會做事，你是來讓團隊更會做事。**
2. **授權不是丟工作，是交出責任與信任。**
3. **團隊要自主運作，前提是安全感。**
4. **帶人不是只帶績效，也要帶心。**
5. **好的主管不是讓大家依賴你，而是讓大家在沒有你時也能運作。**
