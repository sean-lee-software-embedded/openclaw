Project Board Demo

這頁是給 demo 用的入口。你可以直接打開 Board 和 Daily Brief，也可以照著下面的 60 秒講稿走一遍。

Open Board Open Daily Brief
60 秒 demo script
  1. 先打開 Board，展示 Board / Backlog / Sprints / Reports 四個 view。
  2. 在 Board 裡拖動一張卡片，示範 status 流轉。
  3. 點開一張 issue，展示 description / sub-task / comments / assignee / sprint。
  4. 切到 Backlog，示範 issue 可直接移到 sprint。
  5. 切到 Sprints，完成 sprint 並處理 unfinished issues。
  6. 最後打開 Daily Brief，展示每天自動更新的新聞頁。
